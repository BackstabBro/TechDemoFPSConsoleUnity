# TechDemoFPSConsoleUnity

Техническое демо игры от первого лица (FPS), разработанное с упором на чистую архитектуру, гибкую отладку и модульность компонентов.

## Стек и архитектура

* **Внедрение зависимостей (DI)** — архитектура построена на базе IoC-контейнера VContainer для обеспечения слабой связанности компонентов.
* **Реактивное управление** — собственная реализация реактивных параметров (Reactive Properties), состояние которых можно динамически изменять в рантайме.
* **Система сохранений** — локальный модуль автоматического сохранения и загрузки (Save/Load) игрового прогресса и настроек.

## Игровые системы

* **Внутриигровая консоль разработчика** — полноценная отладочная консоль в стиле движка Source, позволяющая на лету менять реактивные параметры игры через команды.
* **Контроллер персонажа** — кастомное перемещение игрока от первого лица с продвинутой физикой, базирующееся на реализации Catlike Coding.
* **Диалоговая система** — интеграция инструмента Yarn Spinner для создания нелинейных ветвящихся диалогов и текстовых сценариев.
* **Система получения урона** — изолированный сервис обработки входящего урона, включающий в себя просчет повреждений от падения с высоты (Fall Damage).
* **Система дней и уровней** — внутриигровые часы с делением игрового процесса на дни. Переход на следующий день (уровень) осуществляется как через консоль, так и при взаимодействии с интерактивными объектами.
* **Система заданий** — базовая квест-система с динамической привязкой пула задач к конкретному дню и реализацией ограничения по времени на их выполнение.
